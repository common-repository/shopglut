== Description ==

= A WooCommerce Enhanced Plugin that Gives You More Flexible and Customized Options (for now wishlist) with ShopGlut =

== **Wishlist Free Version Features** ==
* Require Login Option: Control whether users need to log in to use the wishlist.
* Merge Guest Wishlist After Login: Automatically merge guest wishlist with the user’s account wishlist upon login.
* Product Subscription Notifications: Logged-in users can subscribe to receive notifications about the products on their wishlist.
* Wishlist Layout Customization: Customize the wishlist page, choosing which product information to show (e.g., image, name, price, SKU, etc.).
* Hide Out-of-Stock Products: Option to hide out-of-stock products from the wishlist.
* Side and Popup Notifications: Choose between side notifications or popup notifications.
* Added to Wishlist Notifications: Customizable notifications when a product is added to the wishlist.
* Notification Effects: Use animation effects like fade, slide, or bounce for wishlist notifications.
* Notification Positioning: Set the location of the notification on the screen (e.g., top-right, bottom-left).
* Enable Wishlist on Product Pages: Display the wishlist button on individual product pages.
* Button Customization: Customize the wishlist button with or without icons, and set the text before and after adding a product to the wishlist.
* Button and Icon Customization: Customize the button and icon colors, margins, and padding for different sections of the wishlist page (Product, Shop, Archive).
* Enable Wishlist on Shop/Archive Pages: Display wishlist buttons on category and shop pages.
* Remove Wishlist if Added to Cart: Choose whether to remove wishlist items when added to the cart.

== Wishlist Pro Version Features ==
* Email Scheduling: Automatically send email notifications after a specified time when a product is added to the wishlist. Configure the delay using minutes, hours, or days.
* Cron URL for Automation: Automate email sending with a Cron URL for reliable email execution.
* Enable Multi-Tab Wishlist (MoveList): Users can create and organize products into multiple wishlists (works with logged-in users).
* Manual Email Sending: Manually send emails related to wishlist products with easy-to-use options.
* Enable Wishlist on Account Pages: Display the wishlist as part of the user’s account page with customizable styling.
* Enable MoveList Button on Product, Shop, and Archive Pages: Users can move products between different wishlists directly from the product, shop, and archive pages.

== Installation ==

Installing ShopGlut is just like any other WordPress plugin.

Navigate to your WordPress "Plugins" page, inside your WordPress dashboard, and follow these instructions:

1. In the search field, enter **ShopGlut**. Click "Search Plugins", or hit Enter.
2. Select **ShopGlut** and click either "Details" or "Install Now".
3. Once installed, click "Activate".
